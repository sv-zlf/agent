Executes shell commands and returns the output.

**Usage:**
- Use ONLY when specialized tools are insufficient
- The command parameter is the shell command to execute
- Commands are executed in the current working directory
- Returns stdout and stderr

**Format:**
```json
{
  "tool": "bash",
  "parameters": {
    "command": "npm test"
  }
}
```

**Allowed Use Cases:**
- Running tests (npm test, pytest, etc.)
- Building projects (npm run build, make, etc.)
- Git operations (git status, git commit, git push, etc.)
- Installing dependencies (npm install, pip install, etc.)
- Executing project-specific scripts
- Operations that cannot be completed with other tools

**NOT Recommended For:**
- Searching files - Use Glob tool instead
- Searching file contents - Use Grep tool instead
- Reading files - Use Read tool instead

**Recommendations:**
- Prefer specialized tools over bash when possible
- Test commands locally before using the tool
- Be aware commands are platform-dependent (Windows vs Linux/macOS)
- For complex operations, break into multiple simpler commands

**command** parameter is required - the shell command to execute
