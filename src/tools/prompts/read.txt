Reads a file from the filesystem.

**Usage:**
- Read file content with absolute paths
- Returns content with line numbers starting from 1
- Can specify offset and limit for large files

**Format:**
```json
{
  "tool": "read",
  "parameters": {
    "file_path": "H:/Project/agent/package.json"
  }
}
```

**Options:**
- `offset`: Start line (0-based, default: 0)
- `limit`: Max lines to read (default: 2000)

**Examples:**
```json
{
  "tool": "read",
  "parameters": {
    "file_path": "H:/Project/agent/src/app.ts",
    "offset": 100,
    "limit": 50
  }
}
```

**Recommendations:**
- Always use absolute paths
- Can read any file type including images
- Returns content with line numbers
- Always read before editing or writing to existing files
- Read multiple files in parallel when exploring

**filePath** parameter is required, must be absolute path
