Performs precise string replacement in a file.

**Usage:**
- Requires reading the file first with `read` tool
- Exact string matching - case-sensitive, including all whitespace
- Must match the exact content including indentation and newlines

**Format:**
```json
{
  "tool": "edit",
  "parameters": {
    "file_path": "H:/Project/agent/src/utils.ts",
    "oldString": "const VERSION = '1.0.0';",
    "newString": "const VERSION = '2.0.0';"
  }
}
```

**Multi-line edit:**
```json
{
  "tool": "edit",
  "parameters": {
    "file_path": "H:/Project/agent/src/utils.ts",
    "oldString": "function hello() {\n  console.log('Hello');\n}",
    "newString": "function hello() {\n  console.log('Hello, World!');\n}"
  }
}
```

**Recommendations:**
- Make oldString as unique as possible to avoid wrong matches
- For large changes, use write tool instead
- If edit fails, re-read the file - content may have changed
- Use replaceAll parameter to replace all occurrences

**exact string matching** is required - the oldString must match file content exactly
