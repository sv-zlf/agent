Performs precise string replacement in a file.

**When to use edit vs multiedit:**
- Edit ONE location in ONE file → use `edit`
- Edit MULTIPLE locations in the SAME file → use `multiedit`

**REQUIRED Format:**
```json
{
  "tool": "edit",
  "parameters": {
    "filePath": "H:/Project/agent/src/utils.ts",
    "oldString": "const VERSION = '1.0.0';",
    "newString": "const VERSION = '2.0.0';"
  }
}
```

**Important Notes:**
- Use `filePath` (NOT `filepath`)
- Use `oldString` (NOT `oldtext`)
- Use `newString` (NOT `newtext`)
- exact string matching required - oldString must match file content exactly
