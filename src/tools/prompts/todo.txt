### TodoWrite - Create/Update Task List
Creates or updates a task list to track progress through a multi-step task.

Parameters:
- todos (array, required): List of todo items
  - content (string, required): Task description
  - done (boolean, optional): Completion status
  - id (string, optional): Task identifier

### TodoRead - Read Task List
Reads the current task list from ~/.ggcode/todos.json.

### TodoDelete - Delete Task
Deletes a specific task by ID.

### TodoClear - Clear All Tasks
Clears all tasks from the task list.

Usage:
- Use TodoWrite at the beginning of complex tasks to break them down
- Update tasks as you complete them by setting done: true
- Use TodoRead to check remaining tasks
- Use TodoDelete to remove individual tasks
- Use TodoClear to reset the task list when starting a new task
- Tasks are persisted to ~/.ggcode/todos.json and survive across sessions

Recommendations:
- Break large tasks into smaller, manageable steps
- Update task status as you progress through the work
- Use descriptive task names that clearly indicate what needs to be done
- Review the task list regularly to stay on track
