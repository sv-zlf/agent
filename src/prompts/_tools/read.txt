## read

Reads a file from the local filesystem.

**Usage:**
- Reading file contents to understand code
- Checking configuration files
- Viewing existing files before editing

**Parameters:**
- `filePath` (required): Absolute path to the file
- `offset` (optional): Start line number (0-based, default: 0)
- `limit` (optional): Maximum lines to read (default: 1000)

**Example:**
```json
{
  "tool": "read",
  "parameters": {
    "filePath": "H:/Project/agent/src/index.ts",
    "offset": 0,
    "limit": 50
  }
}
```

**Best Practices:**
- Always use absolute paths
- Read before editing existing files
- Use offset/limit for large files
- Can read any file type including images
